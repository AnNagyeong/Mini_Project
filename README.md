# Mini_Project
방학동안 진행한 미니 프로젝트와 실습에 관한 내용을 기록합니다.  
기존에 사용하던 '코스모스' 어플의 불편한 점을 개선하고, 구현하고 싶은 기능들을 직접 만들어 보는 활동에 대한 내용이 포함됩니다.  


[ 구성 ]  
1. 백엔드 + 약간의 풀스택
2. Linux / Server / Infrastructure

[ 시나리오 ]  
1. 학생이 앱에 로그인
2. 오늘 수업 중 아직 출석하지 않은 수업이 존재함
3. 출석 마감 3시간 전 푸시 알림, 웹 알림 등으로 사용자에게 알림 전송

[ 전체 시스템 구조 ]  

[모바일/웹]  
     ↓  
[백엔드 API 서버]  
     ↓  
[DB]  ←  수업 / 출석 / 사용자 정보  
     ↓  
[알림 스케줄러]  
     ↓  
[푸시]
