# Activity_2 시스템 구성 요소

[ 스케줄러 ]  
- 실행 주기 : 5분
- 방식 : cron
- 현재 시간 기준, 출석 마감 임박 수업 탐색, 알림 대승 선별


[ 출석 체크 조건 ]  
1. 수업 날짜 = 오늘  
2. 출석 상태 ≠ 출석  
3. 출석 마감 - 현재 시간 ≤ 알림 기준 시간  
4. 이미 해당 알림을 보낸 적 없음

[ API ]  
1. 출석 대상 조회 API
2. 알림 발송 API
(DB와 연결하여 사용)
